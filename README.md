# Plant Lectin Families Reference Table

A comprehensive reference table of plant lectin families with conserved Pfam domains, model organisms, and accession numbers for genome-wide lectin identification and phylogenetic analysis.

## Data Formats

This repository provides the lectin reference data in multiple formats:

- **Markdown** (`lectin_reference_table.md`) - Human-readable format
- **CSV** (`lectin_reference_table.xls`) - For spreadsheet applications and data analysis
- **JSON** (`lectin_reference_table.json`) - For programmatic access and APIs

## Applications

- Genome-wide lectin identification using Pfam domains
- Phylogenetic analysis of plant lectin families
- Functional annotation of lectin-like sequences
- Comparative genomics studies

## Data Sources

All accession numbers and Pfam IDs are from public databases:
- **UniProt**: Protein sequences and functional annotation
- **GenBank**: Nucleotide and protein sequences  
- **Pfam**: Protein family database with HMM profiles

## Reference

Van Damme, E. J., Lannoo, N., & Peumans, W. J. (2008). Plant lectins. In Advances in botanical research (pp. 107–209). https://doi.org/10.1016/s0065-2296(08)00403-5
